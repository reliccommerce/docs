How To Install It?

To install a plugin in your WordPress admin panel, follow the following steps:

Plugin Installation Method

    1. Go on your Wordpress Admin

    2. Click "Plugins » Add new"

    3. Click "Upload plugin"

    4. Select the plugin zipped file and validate

    5. Activate the plugin

Plugin Settings

General Setting

  2. 1). WP Fancy Menu Button color and background ?

  1. Go To Dashboard > WP Fancy Menu  
  2. 2). WP Fancy Menu Display Location ?
  3. 1). Select the Display Location Ex: Center Top .
