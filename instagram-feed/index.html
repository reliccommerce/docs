How To Install It?

To install a plugin in your WordPress admin panel, follow the following steps:

Plugin Installation Method

    1. Go on your Wordpress Admin

    2. Click "Plugins » Add new"

    3. Click "Upload plugin"

    4. Select the plugin zipped file and validate

    5. Activate the plugin

Plugin Settings

Relic Instagram Feed

  1. Go To Dashboard > Instagram Feed 
  2. Make your changes like below image
  3. Set Username and API Token
  4. Select layout and other settigns
  5. Select layout and other settigns
  6. Now enter shortcode where you want to display
    shortcode : [relic_instagram_feed]

    Use the function do_shortcode('[relic_instagram_feed]');

ShortCode Combination

  1. You can combine shortcode for every settings like: layout, number_of_column etc
  2. Example : [relic_instagram_feed layout=thumbnail number_of_column=3] or [relic_instagram_feed layout=blog_style number_of_column=4 feed_display_type=loadmore ]
