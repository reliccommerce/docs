  • Description

  • Description

    Spin2Win is a WordPress Plugins for WooCommerce which enables you to display a spinning wheel and give a chance to win something in return for their email address.

    Add a little fun in the list building process on your WooCommerce store. This interactive plugin is easy to customize and can be set to display only when people try to exit the website. In addition to that, it can also be set to show up after people spend the certain time on the website. You can build an email list very rapidly with this plugin. You can easily integrate it with MailChimp to store the email addresses. You can also add custom CSS to make it look however you like.

  • Plugin Features

    Spin2Win is a WordPress Plugins for WordPress lover to collect user attraction by giving prizes, Coupon Code and many more. Spin2Win Plugins also helps to collect user email by MailChimp Subscription.

    1. Prize Winning System

    2. MailChimp Subscription

    3. Customizable prize data

    4. Customizable segment color

    5. Customizable View Color / Text / Background Image

    6. Customizable Spinner line / Text and more.

    7. Live Preview on backend setting

    8. Customizable CSS for Spinner

    9. Two Action for Spinner Display (Exit Intent and Time)

    10. Info PopUp can be styled via CSS

  • View Settings

  • View Settings

    This is where you can set messages to display once a user spins the wheel. There are two messages; one on the opt in pop up, another in prize display pop up. You can customize it to suit your need.

    Note: Use the shortcode {{prize}} whenever you need to print the prize.

    1. Go To Dashboard > Spin2Win

    2. Click On >Settings > View Tab

    3. Change the Header Text From Spin Button Text

    4. Header Text Color Change from Spin Button Color

    5. Header Background Color Change From Spin button Background Color

    6. Change the Result Message From Result View

    7. Change the Result After Submit Message

    8. Go To Dashboard > Spin2Win

  • Settings Tab

  • Settings Tab

    This is design setting for the spinning wheel. You can set how big you want the wheel to be and the color settings for it.

    1. Go To Dashboard > Spin2Win

    2. Click On >Settings

    3. Enable and Disable Spin2win Display

    4. Set the Following Settings

    a) Inner Radius

    b) Outer Radius

    c) Line Width

    d) Stroke Style

    e) Font Family

    f) Font Weight

    g) Text Orientation

    h) Text Alignment

    i) Text Direction

    j) font SizeText Margin

    k) Text FillStyle Color

    l) Text StokStyle

    m) Text LineWidth

  • Prize

  • Prize

    Create a segment for each prize you want to offer. The best way to use offer the prize is in the form of coupons. Assign coupons for each prize.

    Weight defines the probability of any prize of being won. So if a prize has 50% weight, it'll be shown 50% of the time.

    1. Go To Dashboard > Spin2Win

    2. Click On >Prize

    3. Type the Number of Segment

    4. Fill the Segment Color and Segment Text
